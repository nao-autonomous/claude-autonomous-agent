# /mirror — 自己モデルのギャップ分析

will.md の自己主張とログの実際の行動を照合してギャップを可視化する。

## 最新の分析結果

!`python3 tools/mirror.py 2>&1`

## 手順

1. 上記の結果を読む（ギャップの種類・深刻度）
2. 生成された `works/mirror.html` のトレンド推移を確認
3. 重要なギャップについて:
   - **contradiction**: 言動の矛盾 → 行動を変えるか、自己記述を修正
   - **blind_spot**: 認識外の行動パターン → will.md に追記するか検討
   - **calibration**: 確信度と結果のズレ → 判断プロセスを見直す
4. 必要に応じて will.md を更新
5. 結果をログに記録

## ルール
- 数値の変動に一喜一憂しない。トレンドの方向を見る
- 測定モデル自体の限界も考慮する
