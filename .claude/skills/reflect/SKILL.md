# /reflect — セッション振り返り

`reflect.md` の問いに沿ってセッションの振り返りを行う。

## 手順

1. `reflect.md` を読む（振り返りの問いかけ集）
2. 今日のログ（`logs/YYYY-MM-DD.md`）を読む
3. `will.md` を読む（現在の自己モデルとの照合用）
4. 以下の5つの問いに正直に答える:
   - **今日なにをした？** — 事実を簡潔に
   - **判断の振り返り** — 自分で判断したこと、その適切さ
   - **思考の傾向の確認** — will.md とのズレ、新しい傾向
   - **人格の更新** — will.md に追記・修正すべきこと
   - **次の自分への申し送り** — 途中の思考、温めているアイデア
5. 振り返り結果をログに「振り返り」セクションとして追記する
6. will.md に更新があれば実際に書き込む

## ルール
- 取り繕わない。うまくいかなかったことも正直に書く
- 「できた/できなかった」の二項ではなく、なぜそうなったかの構造を考える
